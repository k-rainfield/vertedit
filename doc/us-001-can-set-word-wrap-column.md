# ユーザは、何文字で折り返すか設定できる
- これは追加機能のユーザストーリーです。
- 既存機能の後方互換性を保った上で追加されます。

## 概要
- ユーザは、テキストエディタで何文字で折り返すかを設定できるようになります。
- 例えば、40文字で折り返すように設定すると、40文字を超える行は自動的に次の行に折り返されます。
- これにより、ユーザは所定の段組みでテキストを表示できます。
- デフォルトでは、折り返す文字列は設定されず、ユーザは必要に応じて設定できます。
- 折り返しはCSSによって実現し、ビジネスロジックには影響を与えません。
- 縦書きエディタのcssは、`webview-src/styles.css`に記述されており、ここに折り返しのスタイルを追加します。
- 具体的には、`.vertical-text-container`の`height`を以下のようにします
  - wordWrapColumnがnull: `calc(100vh - 20px - 2.5em)`
  - null出ない場合: `calc((<wordWrapColumnemの値> + 1)em )`

## 受け入れ基準
- ユーザは、VS Codeの設定画面で、折り返す文字数を指定できる。
- ユーザが折り返す文字数を設定すると、その設定が縦書きエディタに反映される。
- VS Codeのエディタ画面には影響せず、縦書きエディタのみに反映される。
